Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

[0.1.4] - 2025-09-20

  • Added basic search through filtering by title to OPDS.

[0.1.3] - 2025-04-14

  • Rewrote OPDS support. It supports pagination and should be faster.

[0.1.2] - 2024-04-15

  • Cleaned up asset handling. Smaller main.js in final container (55 KB vs 65 KB).
  • Details page has nicer styling

[0.1.1] - 2024-02-24

First release with a version number and official documentation website.

Added

  • Show book "comments" (description) on details page
